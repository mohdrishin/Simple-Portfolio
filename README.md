# designerfolio - ⚡️ Simple Portfolio website for Designers

## Features
* Single Page Layout
* Valid HTML5 & CSS3
* Custom SCSS
* Responsive Design
* CSS Grid Portfolio Section
* Custom Animations
* SEO Optimized
